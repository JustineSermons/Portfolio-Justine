Noodletopia - Group Project App Design

Role: Designer

Tools: Figma

URL: Figma Link Figma Link

Date: Fall 2024

Project Description: An app design for noodle lovers. Noodletopia allows you to make an account, see menu options, use reward points for free menu items, write a review, customize noodle bowls, and order food. A simple and clean design was used to help users order with ease and not feel too distracted or overwhelmed. It suits the theme while still looking interesting.

Outcome: Successfully created an app design in Figma with a group for a restaurant based on noodles. I aided in creating the project idea, the design process, and creating various screens.
